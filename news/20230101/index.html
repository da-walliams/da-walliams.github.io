Pilot Version of Personal Website Published

Wally the Boogie is planning to publish a personal website under his real name expecting it to provide a portfolio for his academic career.

He launched this website as a pilot under the pseudonym “Wally the Boogie,” which he has used on social media, to get ready for the release of the official version.

The purpose of this website is to identify what features Wally the Boogie should implement on his personal website.

In other words, this website is a pilot version of the forthcoming official website.

This website is hosted by Github Pages. Wally the Boogie would greatly appreciate it if you could submit an issue or create a pull request in the repository here.
